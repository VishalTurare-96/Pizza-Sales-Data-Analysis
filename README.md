# 🍕 Pizza Sales Data Analysis (SQL + Excel Dashboard)
Pizza Sales Data Analysis using SQL and Excel Dashboard | Data Analytics Portfolio Project


## 📌 Project Overview

This project analyzes **pizza sales data** to extract key business insights using **PostgreSQL for data analysis** and **Excel for interactive dashboard visualization**.

The goal of the project is to simulate a **real-world business intelligence workflow** where raw sales data is transformed into meaningful insights that help stakeholders make better business decisions.

The analysis focuses on identifying:

* Revenue performance
* Sales trends over time

# 📊 Business KPIs Analyzed

The following **Key Performance Indicators (KPIs)** were calculated using SQL.

### 1️⃣ Total Revenue

Total revenue generated from all pizza sales.

```
Total Revenue = SUM(total_price)
```

---

### 2️⃣ Average Order Value

Average amount spent per order.

```
Average Order Value = Total Revenue / Total Orders
```

---

### 3️⃣ Total Pizzas Sold

Total number of pizzas sold across all orders.

```
Total Pizzas Sold = SUM(quantity)
```

---

### 4️⃣ Total Orders

Total number of unique orders placed.

```
Total Orders = COUNT(DISTINCT order_id)
```

---

### 5️⃣ Average Pizzas Per Order

Average pizzas purchased in a single order.

```
Avg Pizzas Per Order = Total Pizzas Sold / Total Orders
```

# 📈 Business Analysis & Insights

## 1. Daily Trend for Total Orders

Analyzed the number of orders placed **per day of the week** to understand peak business days.

**Business Insight**

* Helps determine which days drive the most revenue.
* Useful for staffing and inventory planning.

---

## 2. Hourly Trend for Orders

Analyzed order volume across different hours of the day.

**Business Insight**

* Identifies peak ordering times.
* Helps optimize kitchen operations and delivery staff.

---

## 3. Percentage of Sales by Pizza Category

Calculated how much each pizza category contributes to overall revenue.

**Business Insight**

* Understand which categories drive the most sales.
* Helps prioritize menu items and promotions.

---

## 4. Percentage of Sales by Pizza Size

Determined which pizza sizes generate the most revenue.

**Business Insight**

* Understand customer preference for pizza sizes.
* Optimize pricing strategy.

---

## 5. Total Pizzas Sold by Category

Analyzed total sales volume across different pizza categories.

**Business Insight**

* Identify popular product segments.

---

## 6. Top 5 Best-Selling Pizzas

Identified pizzas with the highest sales volume.

**Purpose**

* Promote these items more aggressively.
* Highlight them in marketing campaigns.

---

## 7. Bottom 5 Worst-Selling Pizzas

Identified pizzas with the lowest sales volume.

**Purpose**

* Evaluate if these should be:

  * Repriced
  * Improved
  * Removed from the menu
